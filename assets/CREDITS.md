# Asset credits

All external art is CC0 (public domain); credits kept here out of courtesy and so the
sources can be re-downloaded.

## Tiles (`assets/tiles/`)

**16x16 Block Texture Set** by ARoachIFoundOnMyPillow — CC0
https://opengameart.org/content/16x16-block-texture-set
(direct file: https://opengameart.org/sites/default/files/blocks_2.zip)

Files used (renamed): dirt, grass (`grass_side`), stone (`stone_generic`), granite,
basalt, obsidian, gravel, snow, mossstone (`cobblestone_mossy`),
ore_nuggets (`stone_generic_ore_nuggets`), ore_crystal (`stone_generic_ore_crystalline`).

These are used as luminance/detail sources only — `TileAtlas.ComposeHybrid` remaps them
onto the game's own per-tile palette at startup.
