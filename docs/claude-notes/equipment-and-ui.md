Character/equipment system added 2026-07-13 (branch noita-sim):

- **Equipment paper-doll** (`Equipment` class in Player.cs, `EquipSlot` enum): 11 slots — Torch, Head, Chest, Legs, Feet, Weapon1/2, MiningTool, Gloves, Accessory1/2. Slots are *pointers into the inventory* (same rule as Toolbelt — equipping never changes counts). RunSave **v15** serializes the slot block after the toolbelt; enum order is the serialization order, append only.
- Armor is now piece-based: chest 40% (old HasArmor value), helm/legs 10%, boots/gloves 5%, aegis_pendant +10% — `Player.DamageTakenMultiplier` reads `Equipment.ArmorReduction`; `HasArmor` bool is vestigial (still saved). Light comes from the torch slot via `Player.EffectiveLightTier` (LightTier remains the crafted-rung recipe gate).
- New craftables: iron/chitin helmet+leggings+boots, leather_gloves / iron_gauntlets (15%/30% faster swings via `Equipment.MineSpeedMul` in `MineCooldownFor`), accessories band_regen (regen tick in Player.Update), magnet_ring (PickupReach 4→16), miners_charm (+1 in EffectivePickaxePower), aegis_pendant. Craft auto-equips (`Wear`/`Trinket` helpers in Game1.Items.cs).
- **CharacterScreen** (src/UI/CharacterScreen.cs, I key): Terraria-styled — beveled indigo slots, ghost icons in empty slots (GhostIds), gold pulsing eligibility rings on hover/drag, rarity-coloured tooltips (`Rarity`), drag-and-drop + RMB quick-equip, backpack grid 8 cols. Intercepts input like the crafting menu; drawn before crafting overlay.
- **Planet godmode** (G / DM_GOD) now also grants 9999 of every material (`GrantGodmodeMaterials` sweeps Tiles.ResourceOrder + meat/hide/chitin + all recipe cost ids).
- **DM_CHARSCREEN=1** opens the character screen on run start (headless screenshot verification with DM_AUTOSHOT; no synthetic keys on macOS).
- **PixelFont** gained '%' and '×' glyphs and folds — – − to '-' — recipe copy with −40%/5×IRON now renders instead of '?'.
